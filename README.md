# Proyecto Validación de Triángulos
![Android CI](https://github.com/wmarcialanzas/TestingSoftware/actions/workflows/android.yml/badge.svg)

## Descripción
Este proyecto utiliza **Kotlin** y **MockK** para validar la lógica de construcción de triángulos con una cobertura del 100%.

## Estado de las Pruebas
La insignia superior indica si los tests están pasando automáticamente en GitHub Actions tras cada cambio.
